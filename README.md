# Projeto_Final_SQL_Credito
Portfolio desenvolvido integrando SQL e Machine Learning para análise exploratória e análise de risco de crédito.

Neste projeto, estaremos fazendo uma análise exploratória dos dados contidos numa planilha Big Data de crédito, de onde estaremos extraindo insights importantes sobre a distribuição dos dados e as relações estabelecidads entre eles.
